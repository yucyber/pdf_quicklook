# 🎯 从这里开始！

欢迎！这是你的 PDF 预览插件项目。

## ⚡ 3 步快速开始

### 方式 1️⃣：Vue3 版本（推荐）

```bash
cd vue3-version
npm install
npm run dev
```

### 方式 2️⃣：原生 JS 版本

```bash
cd vanilla-js-version
npm install
npm run dev
```
